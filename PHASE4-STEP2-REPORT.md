## 🔧 技术实现

### 组件层级

```
App.jsx
└── DiscussionPanel (集成组件)
    ├── MessageTimeline (时间线)
    │   └── MessageBubble (气泡) × N
    └── 输入区 (选择器 + 输入框 + 按钮)
```

### 状态管理

```javascript
const [messages, setMessages] = useState([])      // 消息列表
const [selectedAgent, setSelectedAgent] = useState(null)  // 接收者
const [newMessage, setNewMessage] = useState('')  // 新消息内容
const [messageType, setMessageType] = useState('text')    // 类型
const [stats, setStats] = useState(null)          // 统计
```

### 消息流转

```
用户输入 → 点击发送 → 添加到 messages 数组
                ↓
        MessageTimeline 接收新数组
                ↓
        渲染 MessageBubble 组件
                ↓
        自动滚动到底部
```
