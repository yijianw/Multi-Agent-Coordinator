## 🧪 手动测试步骤

### 1. 界面加载测试
- [ ] 页面正常加载，无报错
- [ ] 标题显示 "📜 Multi-Agent Coordinator"
- [ ] 右上角有 "🚀 启动会话" 按钮

### 2. Agent 配置测试
- [ ] 默认显示 1 个 Agent 配置卡片
- [ ] 点击 "+ 添加" 按钮，新增一个 Agent
- [ ] 点击 "✕" 按钮，移除 Agent (至少保留 1 个)
- [ ] 切换 Agent 类型 (coder/reviewer/tester 等)
- [ ] 输入角色/职责描述
- [ ] 输入功能强化内容

### 3. 工作流模式选择测试
- [ ] 显示 4 种模式卡片 (流水线/并行/讨论/混合)
- [ ] 点击不同模式，正确选中
- [ ] 选择"混合"模式时，显示 DAG 编辑器

### 4. DAG 编辑器测试 (仅混合模式)
- [ ] DAG 编辑器正常显示
- [ ] 可以看到 Agent 节点
- [ ] 可以拖拽节点调整位置
- [ ] 可以从节点拖拽连接线

### 5. 任务描述测试
- [ ] 在任务描述文本框输入内容
- [ ] 内容正确保存

### 6. 启动会话测试 (需要 OpenClaw 服务运行)

**注意:** 此测试需要 OpenClaw 主服务正在运行 (端口 3456)

- [ ] 点击 "🚀 启动会话" 按钮
- [ ] 会话开始运行，按钮变为 "⏹️ 停止"
- [ ] 实时监控面板显示 Agent 状态
- [ ] 日志流开始滚动
- [ ] 完成后显示结果视图
